How to apply a theme: open pwa/index.html, find the tailwind.config = { ... } block, and replace the colors and boxShadow objects with the snippet from the theme you want. Then update the .poppy-wash rule in the <style> block below it. No other code changes are needed — every Tailwind utility class (bg-poppy-500, text-petal-600, etc.) re-renders to the new palette automatically.
